# CrisisMatrix Website

A simple, responsive landing website for **CRISIS MATRIX**, an emergency preparedness and survival guidance app focused on AI-assisted crisis support, offline readiness, SOS features, first aid guidance, and global emergency information.

## Overview

This website acts as the public landing and legal-information page for CrisisMatrix. It presents the brand, app value proposition, key features, and links to legal documents such as the Terms of Service and Privacy Policy.
